SpecialistSteak @ GitHub

Version 1.2.0

Overview

This API is designed to create large (or small) amounts of realistic and believable profiles of fictitious people, primarily intended for testing purposes. It can be a helpful resource for developers who need to quickly generate sample user data in both small and large quantities. Moreover, the data from this API is skewed towards the global and country averages, making it a fairly accurate representation of what real people might look like.


Features

  • Generate random people profiles with realistic data, including names, ages, jobs, addresses, and more.
  • Generate up to 50,000 profiles at once.
  • Generate individual pieces of information, such as names, birthdays, realistic addresses, etc.
  • The service is completely free and requires no signup
